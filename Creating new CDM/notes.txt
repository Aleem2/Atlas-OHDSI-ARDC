How did I verify it 

===========================
The default admin user for the official PostgreSQL Docker container is postgres.

When you run the container without specifying a different user via the POSTGRES_USER environment variable, the container automatically creates a superuser named postgres. This user also has a database created with the same name, postgres, by default.

It is important to note that while the user is set to postgres by default, you must still provide a password for this user by setting the POSTGRES_PASSWORD environment variable. If you don't, the container will not run.
